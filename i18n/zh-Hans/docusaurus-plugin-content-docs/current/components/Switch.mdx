# Switch 开关

开关组件是在移动设备上调整设置的首选方式。开关控制的选项，以及它所处的状态，应该从相应的内联标签中清楚地表明。

## Basic switches

<Wrapper>
  <Grid spacing={2} justifyContent="center">
    <Switch defaultChecked />
    <Switch disabled defaultChecked />
    <Switch />
    <Switch disabled />
  </Grid>
</Wrapper>

```jsx
<Switch defaultChecked />
<Switch disabled defaultChecked />
<Switch />
<Switch disabled />
```

## Label

You can provide a label to the Switch using `label` prop.

<Wrapper>
  <Grid spacing={2} justifyContent="center">
    <Switch label="Switch" defaultChecked />
    <Switch label="Switch" defaultChecked disabled />
  </Grid>
</Wrapper>

```jsx
<Switch label="Switch" defaultChecked />
<Switch label="Switch" />
```

## Size

Use the `size` prop to change the size of the switch.

<Wrapper>
  <Grid spacing={2} justifyContent="center">
    <Switch defaultChecked size="small" />
    <Switch defaultChecked size="medium" />
    <Switch defaultChecked size="large" />
  </Grid>
</Wrapper>

```jsx
<Switch defaultChecked size="small"/>
<Switch defaultChecked size="medium"/>
<Switch defaultChecked size="large"/>
```
