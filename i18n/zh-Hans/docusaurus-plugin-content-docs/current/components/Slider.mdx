# Slider 滑块控件

滑块反映了一根条上的一系列值，用户可以从中选择单个值。 它们通常适用于调节一些设置，譬如调节设备音量、调整屏幕亮度，或者改变图像的滤镜。

## Basic slider

Basic sliders allow users to select a value along a subjective range.

export const Controls = () => {
  const [value, setValue] = useState(0);
  return (
    <>
      <Wrapper>
        <Grid direction="column" spacing={2}>
          <Slider value={value} onChange={(value) => setValue(value)} label />
          <Slider disabled defaultValue={20} />
        </Grid>
      </Wrapper>
    </>
  );
};

<Controls />

## Sizes

For smaller or larger slider, use the `size` prop.

<Wrapper>
  <Grid direction="column" spacing={2}>
    <Slider size="small" defaultValue={10} width={200} label />
    <Slider size="medium" defaultValue={60} width={200} label />
    <Slider size="large" defaultValue={40} width={200} label />
  </Grid>
</Wrapper>

## Discrete sliders

Discrete sliders can be adjusted to a specific value by referencing its value indicator. You can generate a mark for each step with `marks={true}`.

<Wrapper>
  <Grid direction="column" spacing={2}>
    <Slider
      min={20}
      max={200}
      step={20}
      width={200}
      defaultValue={40}
      label
      marks
    />
    <Slider defaultValue={40} step={10} width={200} marks label />
    <Slider
      min={60}
      max={200}
      defaultValue={160}
      width={200}
      step={20}
      marks
      disabled
    />
  </Grid>
</Wrapper>

```jsx
<Slider min={20} max={200} step={20} width={200} defaultValue={40} label marks />
<Slider defaultValue={40} step={10} width={200} marks label />
<Slider min={60} max={200} defaultValue={160} width={200} step={20} marks disabled />
```

## Customization

Here are the example of customizing the component. You can customize the silder by passing `sliderStyle` `thumbStyle` `trackStyle` `railStyle` props.

export const Customization = () => {
  const [color, setColor] = useState("#FF0000");
  return (
    <>
      <Wrapper>
        <Grid direction="column" spacing={2}>
          <Slider
            color="white"
            min={0}
            max={360}
            width={200}
            onChange={(v) =>
              setColor(
                tinycolor({ h: v, s: 1, l: 0.5 }).toHexString().toUpperCase()
              )
            }
            label={color}
            sliderStyle={{
              height: "12px",
              borderRadius: "4px",
            }}
            railStyle={{
              opacity: 1,
              background:
                "linear-gradient(to right, #f00 0%, #ff0 17%, #0f0 33%, #0ff 50%, #00f 67%, #f0f 83%, #f00 100%)",
            }}
            trackStyle={{
              backgroundColor: "transparent",
              border: "none",
            }}
          />
        </Grid>
      </Wrapper>
    </>
  );
};

<Customization />
