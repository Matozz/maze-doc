Dialog

A Dialog is a type of modal window that appears in front of app content to provide critical information or ask for a decision. Dialogs disable all app functionality when they appear, and remain on screen until confirmed, dismissed, or a required action has been taken.

Dialogs are purposefully interruptive, so they should be used sparingly.

Basic dialogâ€‹

Simple dialogs can provide additional details or actions.

Touch mechanics:

  • Choosing an option immediately commits the option and closes the menu
  • Touching outside of the dialog, or pressing Back ( ESC ) keys, cancels the action and closes the dialog
import React, { useState } from "react";
import {
  Button,
  Dialog,
  DialogTitle,
  DialogContent,
  DialogContentText,
  DialogActions,
} from "maze-ui";

export const MyDialog = () => {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button
        label="OPEN DIALOG"
        variant="outlined"
        onClick={() => setOpen(true)}
      />
      <Dialog
        open={open}
        onClose={() => {
          console.log("Dialog Closed");
          setOpen(false);
        }}
        onBackdropClick={() => setOpen(false)}
        exitOnEsc
      >
        <DialogTitle>{"Use Google's location service?"}</DialogTitle>
        <DialogContent>
          <DialogContentText>
            Let Google help apps determine location. This means sending
            anonymous location data to Google, even when no apps are running.
          </DialogContentText>
        </DialogContent>
        <DialogActions>
          <Button
            label="DISAGREE"
            variant="text"
            onClick={() => setOpen(false)}
          />
          <Button label="AGREE" variant="text" onClick={() => setOpen(false)} />
        </DialogActions>
      </Dialog>
    </>
  );
};

Confirm dialogâ€‹

Dialogs allow users to fill out form fields.

Selection: 
import React, { useState, useCallback, useRef } from "react";
import {
  Grid,
  Button,
  Dialog,
  DialogTitle,
  DialogContent,
  DialogContentText,
  DialogActions,
  Label,
} from "maze-ui";

export const MyDialog = () => {
  const [open, setOpen] = useState(false);
  const [result, setResult] = useState("");
  const resRef = useRef();
  const handleSelect = useCallback((e) => {
    resRef.current = e.target.value;
  }, []);

  return (
    <>
      <Grid spacing={2}>
        <Button
          label="DJ MAG VOTE 2022"
          variant="outlined"
          onClick={() => setOpen(true)}
        />
        <Dialog
          open={open}
          onClose={() => console.log("Dialog Closed")}
          width="350px"
        >
          <DialogTitle>{"Best DJ in the world"}</DialogTitle>
          <DialogContent dividers>
            <RadioGroup
              name="dj"
              size="medium"
              orientation="vertical"
              spacing={15}
              onChange={handleSelect}
            >
              <Radio value="Martin Garrix" label="Martin Garrix" />
              <Radio value="Mike Williams" label="Mike Williams" />
              <Radio value="Brooks" label="Brooks" />
            </RadioGroup>
          </DialogContent>
          <DialogActions>
            <Button
              label="CANCEL"
              variant="text"
              onClick={() => setOpen(false)}
            />
            <Button
              label="SUBMIT"
              variant="text"
              onClick={() => {
                setResult(resRef.current);
                setOpen(false);
              }}
            />
          </DialogActions>
        </Dialog>
        <Label>Selection: {result}</Label>
      </Grid>
    </>
  );
};

Scrolling long contentâ€‹

When dialogs become too long for the user's viewport or device, they scroll.

Full-screen dialogsâ€‹

When the fullScreen prop is true, the dialog will show in full-screen.

Full-width dialogsâ€‹

When the fullWidth prop is true, the dialog will adapt to the screen maximum width.

Draggable Dialogâ€‹

You can create a draggable dialog by using react-draggable.

You can use draggable to enable drag behavior on whole dialog, then set the dragOptions and dragFor to specify the components that can be dragged.

import React, { useState } from "react";
import {
  Button,
  Dialog,
  DialogTitle,
  DialogContent,
  DialogContentText,
  DialogActions,
} from "maze-ui";

export const Draggable = () => {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button
        label="OPEN DRAGGABLE DIALOG"
        variant="outlined"
        onClick={() => setOpen(true)}
      />
      <Dialog
        open={open}
        onClose={() => console.log("Dialog Closed")}
        onBackdropClick={() => setOpen(false)}
        draggable
        dragOptions={{
          handle: "#title",
        }}
      >
        <DialogTitle dragFor="title">
          {"Use Google's location service?"}
        </DialogTitle>
        <DialogContent>
          <DialogContentText>
            Let Google help apps determine location. This means sending
            anonymous location data to Google, even when no apps are running.
          </DialogContentText>
        </DialogContent>
        <DialogActions>
          <Button
            label="DISAGREE"
            variant="text"
            onClick={() => setOpen(false)}
          />
          <Button label="AGREE" variant="text" onClick={() => setOpen(false)} />
        </DialogActions>
      </Dialog>
    </>
  );
};
